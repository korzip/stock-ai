# 전체 실행 계획

## A. 현재 상태 요약
- 앱/백엔드/MCP 파이프라인 정상 동작
- LLM 모드(gpt-4.1-mini) 정상
- KR 가격은 pykrx 일부만 동작(티커 리스트 불가)
- DART 공시 적재 정상
- US 일봉(AAPL) 적재/검증 정상

## B. 1차 안정화 (가장 우선)
1) KR 가격 데이터 소스 확정
- 목표: 무료/가벼움 기준으로 대체 소스 결정
- 산출물: 데이터 스펙/스키마 고정

2) KR Top200 적재 안정화
- DART corpCode 기반 리스트 확보 완료
- 가격 소스 확정 후 Top200 재적재

3) 종목 후보 선택 UX
- 후보 3~5개 표시 → 선택 시 자동 확정
- 동명이인/복수 상장 처리

## C. 2차 품질 강화
4) AI 응답 구조화 고도화
- 섹션 분리(요약/가격/리스크/다음 액션)
- UI 카드화

5) 가드레일/정책 강화
- 매수/매도 지시 방지 강화
- 근거 없는 단정 금지

6) 평가 루프 구축
- 대표 질문 세트 구성
- 종목 확정률/환각률 측정

## D. 3차 운영/배포
7) 스테이징 배포
- Cloud Run/ECS 선택
- Managed DB/Redis 구성

8) CI/CD
- PR 머지 → 테스트/빌드/배포
- 마이그레이션 자동화

9) 모니터링
- Sentry + 로그/메트릭
- 헬스체크/알림
